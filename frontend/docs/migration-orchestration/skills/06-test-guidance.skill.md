# Skill: Test Guidance

## Objetivo

Orientar estrategia de testes co-localizados e cobertura minima durante a migracao.

## Checklist de execucao

1. Criar/atualizar testes proximos aos modulos alterados.
2. Cobrir logica de hooks e comportamento de componentes.
3. Priorizar testes de fluxos criticos da feature afetada.
4. Manter consistencia com Vitest + RTL.
5. Medir cobertura e apontar lacunas.

## Saida esperada

- Plano de testes do PR
- Casos obrigatorios por mudanca
- Registro de cobertura e riscos residuais
