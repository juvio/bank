# Skill: PR Planning

## Objetivo

Quebrar a migracao em PRs pequenos, reversiveis e rastreaveis.

## Checklist de execucao

1. Definir objetivo unico do PR.
2. Listar arquivos a criar/alterar/remover.
3. Declarar dependencias de PRs anteriores e seguintes.
4. Definir criterios de aceite mensuraveis.
5. Planejar testes por feature impactada:
   - criar testes quando aplicavel e nao existirem;
   - manter co-localizacao em `__tests__`.
6. Limitar escopo para evitar mistura de temas.

## Saida esperada

- Plano de execucao do PR
- Lista de riscos e mitigacoes
- Sequencia recomendada de commits
