# Prompt: PR Handoff

Prepare handoff do PR [N] para revisao do time.

Entregue:

1. Contexto e objetivo do PR.
2. Mudancas chave por modulo.
3. Dependencias para PRs seguintes.
4. Resultado de lint, build e testes.
5. Riscos residuais e pontos de atencao.
6. Localizacao de todos os arquivos de documentacao.
