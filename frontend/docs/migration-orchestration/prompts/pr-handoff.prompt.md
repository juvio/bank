# Prompt: PR Handoff

Prepare handoff do PR [N] para revisao do time.

Entregue:

1. Contexto e objetivo do PR.
2. Mudancas chave por modulo.
3. Dependencias para PRs seguintes.
4. Resultado de lint, build e testes.
5. Testes criados/atualizados por feature e localizacao em `__tests__`.
6. Riscos residuais e pontos de atencao.
7. Localizacao de todos os arquivos de documentacao.
8. Confirmacao de uso da skill `feature-module-migration` quando o PR for de
   migracao de modulo.
