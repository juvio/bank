# Prompt: PR Execution

Execute o PR [N] da migracao de arquitetura.

Passos:

1. Releia objetivo e escopo do PR no plano.
2. Implemente as mudancas no frontend.
3. Rode validacoes: lint, build e testes.
4. Corrija regressos identificados.
5. Gere documentacao obrigatoria em `frontend/docs/PR[N]`.

Saida:

- Resumo tecnico da implementacao.
- Lista de arquivos alterados.
- Resultado das validacoes.
- Riscos residuais.
