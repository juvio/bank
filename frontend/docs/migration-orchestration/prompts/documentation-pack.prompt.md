# Prompt: Documentation Pack

Gere o pacote documental do PR [N] em `frontend/docs/PR[N]`.

Arquivos obrigatorios:

1. `PR[N]-READY.txt`
2. `README-PR[N].md`
3. `PR[N]-SUMMARY.md`
4. `PR[N]-CHECKLIST.md`
5. `PR[N]-TEST-GUIDE.md`
6. `PR[N]-COMMIT-INSTRUCTIONS.md`
7. `PR[N]-VISUAL-MAP.md`
8. `PR[N]-INDEX.md`

Cada arquivo deve conter informacao coerente com o codigo entregue no PR [N].
